# Smart Notes (External Next.js)

Standalone extraction of Smart Notes with local DB/storage.

## Run

```bash
cd external-apps/smart-notes-next
npm install
./scripts/copy-db.sh
npm run build
npm run start
```

## Register

```bash
./scripts/register-external-smart-notes.sh
```

Open:
- http://localhost:3000/apps/smart-notes-external
