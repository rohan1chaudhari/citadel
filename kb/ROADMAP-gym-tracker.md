# Roadmap — Gym Tracker

## v0.1 (DONE)
- Minimal logging: date, exercise, sets, reps, weight
- List recent entries
- Storage touch on write (for isolation testing)

## v0.2 (Usability)
- Edit + delete entries
- Exercise autocomplete (distinct exercises)
- Basic filters (exercise, date range)

## v0.3 (Progress)
- PR tracking (best weight/reps per exercise)
- Simple charts (volume over time)

## v0.4 (Routines)
- Routine templates
- “Start workout” session flow (optional)
